How to transcribe Zoom Meetings?

To transcribe audio from Zoom, you have two options:


Option 1: Share System Audio (Best for Web Zoom)
  1. Click Share System Audio above.
  2. Select the tab or window playing the Zoom meeting.
  3. Ensure "Share tab audio" is checked.

Option 2: Virtual Audio Cable (Best for Zoom App)
  1. Install a Virtual Audio Cable (e.g., VB-Cable or BlackHole on Mac).
  2. In Zoom Settings > Audio, set "Speaker" to the Virtual Cable.
  3. In this app, select the Virtual Cable as the Input Device.
  4. Crucial: Set the Virtual Cable as your computer's System Default Input if transcription doesn't start.
